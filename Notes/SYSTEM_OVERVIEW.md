# SYSTEM_OVERVIEW

## Purpose
This system is a simple web-based note-taking application designed for a single user.

## System Scope

### In Scope
- Create notes
- Update existing notes
- Delete notes
- View all created notes
- Persist notes in storage

### Out of Scope
- User authentication or authorization
- Multi-user support
- Collaboration features
- Offline support
- Mobile native applications

## System Characteristics
- Single-user system
- Web-based application
- No login or identity management

## Assumptions
- The system is used by only one user
