# User Stories

## Core Functionality

- **As a user**, I want to create a note by pressing a button, **so that** I can store information quickly.

- **As a user**, I want to edit an existing note, **so that** I can update its content when needed.

- **As a user**, I want to delete a note, **so that** I can remove information that is no longer relevant.

- **As a user**, I want to view all my notes, **so that** I can easily review what I have written.

## User Interface

- **As a user**, I want to see all available action buttons (create, edit, delete, history), **so that** I can understand what actions are possible.
